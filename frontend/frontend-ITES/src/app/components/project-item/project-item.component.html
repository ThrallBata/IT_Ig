Интернет-магазин плюшевых игрушек
Сайт интернет-магазина плюшевых игрушек для ООО «ПлюшМиш» в г. Томск. Выполнен в светлых тонах розово-белых оттенков. Сайт обладает следующими функциональными возможностями: Просмотр каталога товаров; Поиск по каталогу товаров; Фильтрация товаров по категориям (размеры, материал изготовления, производитель, наличие товара в магазине); Корзина с выбранными товарами; Форма оформления заказа на сайте; Система оплаты через различные платежные системы (Мир, Visa, MasterCard).
